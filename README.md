👟 Analysis of Shoe Company Strategies in India
📊 Overview

This project analyzes the business strategies of leading footwear companies in India. It focuses on how different brands compete in terms of pricing, marketing, distribution, and customer targeting in the Indian market.

🎯 Objectives
Compare strategies of different shoe brands in India
Understand pricing and positioning techniques
Analyze marketing and branding approaches
Study customer segmentation and market trends
🏢 Brands Analyzed
Nike
Adidas
Puma
Bata
Relaxo
🛠️ Tools & Technologies
Python 3
Google Colaboratory / Visual Studio Code
Pandas, NumPy (for data analysis)
Matplotlib / Seaborn (for visualization)
⚙️ How to Run the Project
Open any Python-supported platform such as Google Colab or VS Code.
Download or clone this repository.
Open the project file (.ipynb or .py).
Upload or insert your dataset file (CSV/Excel format).
Ensure the dataset file name matches the name used in the code.
Run all cells or execute the script.
Analyze the output and results generated.
📂 Data Requirements
Dataset should be in CSV or Excel format
Must include relevant fields such as brand, price, category, etc.
Keep the dataset in the same folder or upload it in Colab
📈 Key Insights
Global brands focus on premium branding and lifestyle appeal
Indian brands dominate in affordability and mass market reach
Digital marketing and e-commerce play a major role in growth
Influencer marketing and collaborations impact brand visibility
